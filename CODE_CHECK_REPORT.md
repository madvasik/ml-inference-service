### Запуск тестов
**В Docker контейнере:**
```bash
docker-compose exec backend pytest tests/unit/ -v
docker-compose exec backend pytest tests/integration/ -v
```

**Локально (требуются зависимости):**
```bash
pytest tests/unit/
pytest tests/integration/
```

**Примечание:** Тесты и скрипты не копируются в Docker контейнер по умолчанию (это нормально для production). Для запуска тестов в контейнере нужно либо:
1. Монтировать папки через volumes в docker-compose.yml
2. Или запускать тесты локально с установленными зависимостями
